Optimalizační hrátky
====================

Co je vaším úkolem
------------------

1.) Nastudovat si optimalizační [volby překladače gcc](https://gcc.gnu.org/onlinedocs/gcc/Optimize-Options.html).

2.) Makefile obsahuje proměnou OPT_LEVEL nastavenou na 0 - voláním "make opt OPT_LEVEL=1" uděláte výstup pro úroveň 1.

3.) Výstupy se jmenují dle zvolené úrovně.

4.) Dále kód upravujte a zkoušejte, co všechno vám jednotlivé úrovně optimalizace z kódu "vyžerou".

5.) Do diskuse nahrajte vše zajímavé, na co jste přišli.
